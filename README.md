This is an all-in-one board for building DIY robots capable of mapping
the environment and self-driving using the map.

Think of this PCB as a miniature robot “motherboard” connecting with 
- A 2D laser distance (LiDAR) scanning sensor
- Two brushless encoder motors (12V or 24V)
- An ESP32 DOIT DevKit v1 30-pin compatible board with WiFi communication
- A battery
